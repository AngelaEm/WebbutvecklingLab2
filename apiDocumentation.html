Glammor

API Documentation

API for creating, reading, updating, and deleting earrings from the Glammor database.

Base URL

URL: [localhost:3000]

Endpoints

[GET] /earrings

Retrieve all earrings from the database.

[GET] /earring/{id}

Retrieve a single earring by its ID from the database.

Parameter: int id

Sample Response:


[DELETE] /earring/{id}

Delete an earring by its ID from the database.

Parameter: int id

[POST] /earring

Create a new earring.

Body:
  • Name: string
  • Description: string
  • Image: URL string
  • Price: float
  • Quantity: int

[PUT] /earring/{id}

Update an existing earring by its ID.

Body:
  • Name: string
  • Description: string
  • Image: URL string
  • Price: float
  • Quantity: int
